About Me

I am a computer science student at Southern New Hampshire University completing my Bachelor of Science in Computer Science. Over the past two and a half years, I have built a strong foundation in software testing and quality assurance, object-oriented design principles, data structures and algorithm optimization, secure coding practices, and database integration.

The CS-499 capstone has been the culminating experience of my academic journey. Starting from a basic contact management service built in CS-320, I systematically enhanced it across three categories: software design and engineering, algorithms and data structures, and databases, transforming it from a simple ArrayList-based prototype into a production-ready application with professional architecture, optimized performance, and enterprise-grade security.

Through the software design enhancement, I refactored the application into a three-tier architecture with controller, service, and repository layers, implemented the Repository and Singleton design patterns, built a centralized validation service, and added professional logging throughout. Through the algorithms enhancement, I replaced ArrayList storage with a HashMap, TreeMap, and LinkedHashMap combination, achieving O(1) lookup performance and demonstrating measurable 100x speed improvements through benchmarking. Through the database enhancement, I integrated SQLite persistence using JDBC, implemented prepared statements to prevent SQL injection, built an audit trail for compliance and security monitoring, and added transaction management with rollback capabilities.

Collaborating with teammates, communicating technical decisions clearly, and approaching security as a first-class concern are professional values that emerged through this process. I am seeking backend software engineering roles where I can apply these skills to build reliable, secure, and maintainable systems.

Three-Category Enhancement

Enhanced Artifacts

The Contact Service was enhanced across all three required categories. Each enhancement builds upon the previous, creating a cohesive, production-ready application.

Enhancement One

Software Design & Engineering

Refactored from flat structure to three-tier architecture. Implemented Repository and Singleton patterns. Separated validation logic, added professional logging, and organized code into meaningful packages.

Key outcome: O(1) HashMap operations, SOLID principles, layered architecture.

Narrative Code
Enhancement Two

Algorithms & Data Structures

Integrated TreeMap for sorted access and range queries, and LinkedHashMap as an LRU cache with automatic eviction. Benchmarked ArrayList vs. HashMap performance across dataset sizes.

Key outcome: 100× faster lookup, sorted access, caching strategy.

Narrative Code
Enhancement Three

Databases

Integrated SQLite database via JDBC for persistent storage. Used prepared statements to prevent SQL injection, built a full audit trail, and implemented transaction management with rollback.

Key outcome: Production-ready persistence, ACID compliance, security-first design.
